## Backend dependency

`leo-api` **alpha.6+** OTP contract:
- `POST /auth/verify-email { email, code }`
- `POST /auth/resend-verify { email }`
- `POST /auth/reset-password/verify { email, code }` тЖТ `reset_ticket`

Run against staging with alpha.6+ API.

### Personal funnel

- [ ] `/signup` тЖТ Personal тЖТ details тЖТ submit
- [ ] Router redirects to `/verify-email` (no imperative push)
- [ ] Enter OTP from email тЖТ session minted
- [ ] Router тЖТ `/onboarding/personal` when `onboardingRequired`
